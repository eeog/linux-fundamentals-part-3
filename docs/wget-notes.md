# Downloading Files With wget

`wget` is a command‑line utility used to download files from the web using HTTP/HTTPS. It is extremely useful when working on remote systems or servers without a graphical interface.

---

## 📥 Basic Usage

```bash
wget https://example.com/file.txt
```

This downloads the file into the current directory.

---

## 🧰 Common Use Cases

- Downloading logs or evidence  
- Pulling scripts or tools during investigations  
- Retrieving remote resources without a browser  
- Automating downloads in scripts  

---

## 🛡️ Security Relevance

Analysts frequently use `wget` to:

- Retrieve suspicious files for analysis  
- Pull down threat intel feeds  
- Download tools onto compromised hosts  
- Collect logs from remote systems  

It’s a fundamental tool for Linux‑based investigations.
